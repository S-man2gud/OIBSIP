# Task 1 - Exploratory Data Analysis (EDA) on Retail Sales Data
This project focuses on performing exploratory data analysis (EDA) on a retail sales dataset to uncover patterns, trends, and actionable insights that can help a retail business make informed decisions.

## Project Objective
The goal of this project is to:

- Understand sales trends over time.
- Analyze customer demographics and purchasing behaviors.
- Identify top-performing product categories.
- Provide actionable recommendations for business growth.

## Dataset Overview
The dataset contains 1,000 records with the following columns:

- Transaction ID: Unique identifier for each transaction.
- Date: Date of the transaction.
- Customer ID: Unique identifier for each customer.
- Gender: Gender of the customer.
- Age: Age of the customer.
- Product Category: Category of the purchased product.
- Quantity: Quantity of items purchased.
- Price per Unit: Price of a single unit.
- Total Amount: Total amount spent in the transaction.

## Key Features
1. Data Cleaning:
- Checked for missing values and duplicates.
- Ensured data consistency and prepared it for analysis.
2. Descriptive Statistics:
- Summarized data to compute key metrics such as mean, median, and standard deviation.
3. Time Series Analysis:
- Identified sales trends over time using monthly sales data.
4. Customer Demographics Analysis:
- Analyzed sales distribution by age and gender.
5. Product Performance Analysis:
- Identified top-performing product categories.
6. Visualization:
- Used bar charts, line plots, and heatmaps to present insights effectively.
7. Recommendations:
- Drafted actionable suggestions based on the findings.
